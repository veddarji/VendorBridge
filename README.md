# VendorBridge

This is a monorepo for the VendorBridge hackathon project.

## Structure
- `/frontend`: React application (Vite, JavaScript)
- `/backend`: Spring Boot application (Maven, Java 21)

## Running Locally

### Frontend
```bash
cd frontend
npm run dev
```

### Backend
```bash
cd backend
./mvnw spring-boot:run
```
